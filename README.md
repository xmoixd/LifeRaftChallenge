# LifeRaftChallenge

dependencies
- **Nodejs**
- **Docker**

setup
there are two ways to start working with this project

1st
 - **npm install**
 - **docker-compose up**

2nd
 - **npm run start**


Usage
 - There is a file named **LifeRaftChallengeMoises.postman_collection.json**
  you can use to import the api to your postman and play around with it
